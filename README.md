# Webpack-101

## Principais comandos

Inicializa o *package.json*
```
npm init -y
```

Instala as dependências listadas no *package.json*

```
npm i
```


Compila o Webpack
```
npm start
```

Compila o Webpack e executar o arquivo gerado no */dist*
```
npm start && node dist/main.js
```
